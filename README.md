## ğŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (XAMPP, WAMP, or similar) for full functionality
- Text editor or IDE (VS Code, Sublime Text, etc.)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/charulekh-prakashan.git
   cd charulekh-prakashan
   ```

2. **Set up local server**
   - If using XAMPP/WAMP: Copy the project folder to `htdocs` or `www` directory
   - Start Apache server

3. **Access the website**
   - Open your browser and navigate to:
     - `http://localhost/charulekh-prakashan/` (for XAMPP/WAMP)
     - Or simply open `index.html` in your browser for static preview

### Configuration

- Update contact information in `contact.html`
- Modify company details in `about.html`
- Add your logo to `assets/img/logo.png`
- Configure database connection (when backend is integrated)
